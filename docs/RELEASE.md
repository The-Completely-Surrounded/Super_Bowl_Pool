# Release Notes (Template)

Use this when publishing a GitHub release.

## Title

`Super Bowl Pool v1.0`

## What is it?

A simple Super Bowl squares pool that runs on any PHP web host. No database setup required.

## Who is it for?

PTA groups, schools, churches, sports clubs, or anyone running a fundraiser.

## Quick Install

1. Download the release ZIP
2. Upload the `release_package/` folder contents to your web host
3. Run `setup.php`
4. Delete `setup.php`
